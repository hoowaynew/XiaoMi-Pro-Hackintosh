# 改善屏幕色彩文件

[English](README.md) | [中文](README-CN.md)

## 注意：仅限BOE NV156FHM-N61 LCD Panel (小米笔记本Pro默认)

## 功能

由Spyder5校准

sRGB 84% -> 97,1


## 安装

1. 拷贝 `NV156FHM-N61.icm` 到 `~/资源库/ColorSync/Profiles`

2. 打开 `系统偏好设置 -> 显示器 -> 颜色`

3. 选择 `NV156FHM-N61` 描述文件

4. 重启
